# Proje Hakkında
Bu uygulama insanların yedikleri meyvelerin faydalarını görebilmesi için yapılmıştır. Kullanıcı yediği meyvenin vitaminini,meyve familyasını ve meyvenin genel özelliklerini öğrenebilir.
